# Kaggle_Chess_AI_Challenge
 
### 제약 조건
1. 5 MiB of RAM*
2. Dedicated CPU: a single 2.20GHz core
3. 64KiB compressed submission size limit
* RAM constraint is not exact. Expect an update which more strongly enforces this limit.

### 목표
 1. 강화학습을 사용
 2. 제약조건을 충족하는 모델 개발
